# 🤖 AI Emotion Detection Robot

A Real-Time AI Emotion Detection System built using Python, OpenCV, FER, Streamlit, and Text-to-Speech.

This project detects human emotions from live webcam video, displays emotion emojis, and speaks motivational or emotional guidance based on detected emotions.

---

# 🚀 Features

✅ Real-Time Face Detection  
✅ Emotion Detection using AI  
✅ Live Webcam Capture  
✅ Emotion Emojis  
✅ Voice Response System  
✅ Motivational Sentences  
✅ Streamlit User Interface  
✅ Offline Emotion Detection  
✅ Happy / Sad / Angry / Fear / Neutral Detection  

---

# ⚙️ Installation

Install all required libraries:

```bash
pip install numpy==1.26.4
pip install pandas==2.2.2
pip install opencv-python==4.10.0.84
pip install fer==22.4.0
pip install tensorflow
pip install streamlit
pip install pyttsx3
pip install pillow
```

---

# ▶️ Run Project

Open terminal inside project folder and run:

```bash
streamlit run app.py
```
